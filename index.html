Comment résoudre une équation du type Ax² + Bx + C = 0 ?

Pour ce faire, il faut calculer le discriminant appelé DELTA telle que DELTA soit égale à B² - 4*A*C.
Selon son signe, 3 cas peuvent se présenter:

  • Si DELTA > 0, alors l'équation admet 2 racines distincts.
  • Si DELTA < 0, alors l'équation admet une racine double.
  • Si DELTA = 0, alors l'équation n'admet aucune solution.
